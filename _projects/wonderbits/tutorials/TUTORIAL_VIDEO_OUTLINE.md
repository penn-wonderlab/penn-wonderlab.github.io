## Production Notes

### Recommended Order for Recording
1. Part 1 (all) - Essential for all users
2. Part 2.1-2.2 - Important organizing features
3. Part 3.1-3.2 - If targeting classrooms
4. Part 2.3-2.4 - Views and Document mode
5. Part 4 - AI features (if enabled)
6. Part 5-6 - Advanced features

### Video Style Guidelines
- Keep videos under 3 minutes when possible
- Use screen recordings with voice-over
- Highlight cursor/clicks
- Use consistent zoom level for readability
- Consider captions for accessibility

### Suggested Tools
- Screen recording: OBS, Loom, or ScreenFlow
- Editing: DaVinci Resolve (free) or Premiere
- Thumbnails: Figma or Canva
